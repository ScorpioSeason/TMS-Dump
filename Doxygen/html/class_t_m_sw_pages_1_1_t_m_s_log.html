TMSwPages.TMSLog.TMSLog ( string  nUnparsed)

Constructor for the TMSLog class.

This takes in a specifically formatted string of BSV (Bar separated values) and parses it to create a new log object. If the string isn't parsed correctly, a new log is created which acknowledges this.
