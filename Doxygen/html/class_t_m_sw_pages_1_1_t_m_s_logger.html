Detailed Description

This class runs the logging functionality for all files in this solution.

This is a static class containing a list of logs and the methods to create new logs. The main function of this class is to call the TMSLog class.

This function will change where the log file is stored.

This function allows the user to select a new location for the log file storage. Then the program copies the current file to the new location. If the copy is successful, the old file is deleted. Otherwise the old file is kept.
